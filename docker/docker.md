# Docker manuals

~~~~
docker run <image-name>

docker image ls

## List Docker containers (running, all, all in quiet mode)

~~~~
docker container ls
~~~~

~~~~
docker container ls --all
~~~~

~~~~
docker container ls -a -q
~~~~


## List Docker images

~~~~
docker image ls
~~~~


## Build images

~~~~
docker build -t <tag-name>
~~~~
