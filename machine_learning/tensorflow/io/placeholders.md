## `tf.placeholder`

- https://www.tensorflow.org/api_docs/python/tf/placeholder
- https://www.tensorflow.org/api_guides/python/reading_data#Feeding

Inserts a placeholder for a tensor that will be always fed.

~~~~
tf.placeholder(
    dtype,
    shape=None,
    name=None
)
~~~~
