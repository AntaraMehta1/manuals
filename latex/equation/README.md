# Equations in latex


## Equation environments


### `equation`

~~~~
\begin{equation}
    f(x) = 3x^{2} + 2x + 1
\end{equation}
~~~~


### `aligned`

~~~~
\begin{aligned}
    f(x) &= 3x^{2} + 2x + 1 \\
    g(x) &= 5x + 1
\end{aligned}
~~~~


### `gather`

~~~~
\begin{gather}
\end{gather}
~~~~


### `gathered`

~~~~
\begin{equation}
\begin{gather}
    f(x) = 3x^{2} + 2x + 1 \\
    g(x) = 5x + 1
\end{gather}
\end{equation}
~~~~
