# Matplotlib commands

## Basics

Import matplotlib:

~~~~
import matplotlib.pyplot as plt
~~~~


Switch backend, e.g. to PDF:

~~~~
plt.switch_backend('PDF')
~~~~

Save figure

~~~~
plt.savefig( 'fig.pdf' )
~~~~
