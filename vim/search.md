#  Search in `vim`

- `/`: forwards search
- `?`: backwards search

- `*`: search for the expression under the cursor

- `n` - next match
- `N` - previous match 

Ignore the case of the letter:
~~~~
:set ignorecase
~~~~

Distinguish the case:
~~~~
:set noignorecase
~~~~

Disable the highlighting:
~~~~
:nohlsearch
~~~~
