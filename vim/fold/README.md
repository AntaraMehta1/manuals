# Folding in `vim`

- `zR`: open all folds
- `zo`: fold open
- `zc`: fold close
- `zr`: fold reduce
- `zm`: fold more
- `zM`: close all fold

If you
~~~~
:set foldmethod=syntax
~~~~

the folds will be specified from the syntax definitions.

If you prefer you can
~~~~
:set foldmethod=indent
~~~~
to have the indentation define the folds.
